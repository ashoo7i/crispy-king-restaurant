# تصميم نظام حماية لوحة الإدارة بكلمة مرور متغيرة (Admin Password Protection)

تصميم وتطوير بوابة أمان للوحة إدارة مطعم **كرسبي كينج** تطلب كلمة مرور للدخول، وتتيح للمشرف تغييرها في أي وقت، مع الحفظ المتزامن في قاعدة البيانات والنسخ الاحتياطي المحلي في المتصفح.

---

## 🎯 الأهداف والميزات (Goals & Features)

1. **بوابة دخول آمنة (Login Gate):** واجهة مستخدم متميزة تحتوي على حقل إدخال كلمة المرور مع تأثيرات بصرية جذابة وحركة اهتزاز (Shake) عند إدخال رمز خاطئ.
2. **تحديث كلمة المرور (Change Password):** إمكانية تغيير كلمة المرور من داخل لوحة التحكم بعد التحقق من الرمز القديم.
3. **الدعم الهجين (Hybrid Support):**
   - **الوضع المتصل (Online Backend):** حفظ وتحديث كلمة المرور في جدول الإعدادات (`Setting`) بقاعدة بيانات SQLite عبر مسارات API جديدة.
   - **الوضع المنفصل (Offline Fallback):** تخزين كلمة المرور محلياً في `localStorage` بالمتصفح لتشغيل وحماية لوحة التحكم مباشرة على استضافة Vercel دون الحاجة لباك إند خارجي.
4. **افتراضية الدخول:** كلمة المرور الافتراضية عند التشغيل لأول مرة هي `admin123`.

---

## 🛠️ التغييرات المقترحة (Proposed Changes)

### 1. قاعدة البيانات (Prisma DB Schema)
إضافة جدول إعدادات مبسط لتخزين القيم العامة مثل كلمة مرور الإدارة:

```prisma
model Setting {
  key   String @id
  value String
}
```

### 2. الباك إند (Express Server - `server/src/routes.ts`)
إضافة المسارات التالية:
- `POST /api/admin/login` - للتحقق من صحة كلمة المرور المدخلة.
- `POST /api/admin/change-password` - لتحديث كلمة المرور في قاعدة البيانات.

### 3. الفرونت إند (React Frontend)
- **بوابة الدخول:** إضافة شاشة طلب كلمة المرور كبوابة قبل الدخول إلى مكون `AdminDashboard`.
- **نموذج التغيير:** إضافة نافذة منبثقة (Modal) داخل لوحة التحكم تتيح للمستخدم تغيير كلمة المرور.
- **إدارة الجلسة (Session Control):** استخدام `sessionStorage` للحفاظ على حالة تسجيل الدخول طالما أن التبويب مفتوح، وزر **"تسجيل الخروج"** لقفل اللوحة يدوياً.

## 🧪 التحقق والاختبار (Verification Plan)

### الاختبار اليدوي
1. فتح لوحة الإدارة والتأكد من ظهور بوابة الدخول وعدم القدرة على تصفح الطلبات بدون كلمة المرور.
2. تجربة الدخول بكلمة مرور خاطئة والتأكد من إظهار رسالة خطأ مع حركة اهتزاز الحقل.
3. الدخول بكلمة المرور الافتراضية `admin123`.
4. الدخول إلى شاشة تغيير كلمة المرور وتغييرها بنجاح.
5. تجربة تسجيل الخروج والتأكد من قفل اللوحة وطلب كلمة المرور الجديدة للدخول مجدداً.
